Setup Guide

Copy these steps to run Sharky locally.

  1. Download the CustomBot folder from the repo.
  2. Create a Discord Application on the Discord Developer Portal.
  3. Insert your bot token into config.py.
  4. Set your IDs: SERVER_ID, GREET_CHANNEL_ID, OWNER_ID.
  5. Invite the bot and run main.py.
