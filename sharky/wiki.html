Welcome to the Sharky Wiki 🦈

Sharky (formerly YouShadeBot) is a Discord bot built in Python that focuses on fun, control, and flexibility. This wiki covers all available features, setup steps, and contribution details.

💻 Current Features

These are all the features that Sharky currently offers.

  • 🙍 Greet user via Command
  • 👋 Automatic Greet Message
  • ℹ️ Info Command
  • 🎮 Bot Activity Changer
  • ⌛ Ping Command
  • 🛑 Moderation Tools
  • 🪨✂️📄 Rock-Paper-Scissor Game
  • 💻 Terminal Control (shutdown, reload, etc.)

🧩 Planned Features

  • 📣 Announcements
  • 🤫 Whisper Messages
  • 📒 Notes System

🚀 Setup Guide

Follow these steps to self-host and run Sharky on your machine.

  1. Clone the repository 
    git clone https://github.com/wollydev24/sharky
  2. Create a Discord Application on the Discord Developer Portal.
  3. Rename config_example.py to config.py and insert your bot token.
  4. Set all required IDs:
    • server → your server ID
    • greet → welcome channel ID
    • owner → your Discord ID
  5. Invite the bot to your Discord server.
  6. Run the bot: 
    py main.py
