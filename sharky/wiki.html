Introduction

1. Welcome to the Sharky Wiki 🦈

Setup Guide

3. Setup Guide

  1. Clone the repository 
    git clone https://github.com/wollydev24/sharky
  2. Create a Discord Application on the Discord Developer Portal.
  3. Rename config_example.py to config.py and insert your bot token.
  4. Set all required IDs:
    • server → your server ID
    • greet → welcome channel ID
    • owner → your Discord ID
  5. Invite the bot to your Discord server.
  6. Run the bot: 
    py main.py
